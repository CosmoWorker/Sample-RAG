# Typescript-only RAG Application

## Tech Stack
* Frontend - React
* Backend - Typescript (Bunjs)
* Database Provider - Supabase
* Vector DB - Supabase (in-built `pgvector` extension)
* ORM - Prisma

## Tools
* Azure OpenAI Text Embedding Models
* Groq - Image Recognition (llama) & Chat (gpt-oss)
* pdf-parse for PDFs
* Cloudinary - File Storage
